 Restoran Otomasyon Sistemi

Bu proje, bir restoranın sipariş alma, masa yönetimi,gelir gider yönetimi ve personel yönetimi işlemlerini kolaylaştırmak amacıyla C# dili kullanılarak geliştirilmiş bir otomasyon sistemidir.

  Kullanılan Teknolojiler

- C# (Windows Forms )
- .NET Framework / .NET Core
- SQL Server (Veri Tabanı)
- Visual Studio

 Özellikler

-  Masa Seçimi ve Takibi  
-  Sipariş Alma ve Güncelleme  
-  Fiyatlandırma ve Hesaplama  
-  Kullanıcı Rolleri (Garson, Müşteri, Yönetici)  
-  Ürün Yönetimi (Menü Ekleme, personel ekleme, gelir/gider yönetme)  
-  Giriş Paneli (Rol Bazlı Giriş)



 Kurulum ve Çalıştırma

Bu projeyi Visual Studio ile açın.Herhangi bir eklenti kurulumu olmadan çalıştırabilirsiniz.
